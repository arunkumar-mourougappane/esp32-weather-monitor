# Release Notes — v1.0.1

This is a minor patch release focused entirely on improving developer experience, code maintainability, and compilation hygiene.

## 📝 What's New

* **Comprehensive API Documentation:** All 9 core header files across the `lib/` directory have been rewritten with full Doxygen-style annotations. This includes `@file`, `@class`, `@brief`, `@param`, and `@return` documentation blocks for every exposed class, struct, and method in the framework.
* **Standardised Include Guards:** Replaced non-standard `#pragma once` compiler directives with historically robust `#ifndef` / `#define` / `#endif` include guards across all `lib/` header files to guarantee clean object linking across the Xtensa toolchain.
